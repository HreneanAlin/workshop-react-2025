## ğŸ“š Exercises

This workshop includes 5 progressive exercises:

1. **Exercise 1: Components & JSX**

   - Learn to create reusable React components
   - Understand JSX syntax
   - Pass data using props

2. **Exercise 2: Props & State**

   - Understand the difference between props and state
   - Use the `useState` hook
   - Update component state

3. **Exercise 3: Event Handling**

   - Handle user interactions
   - Implement shopping cart functionality
   - Work with complex state

4. **Exercise 4: Lists & Keys**

   - Render lists of items
   - Understand the importance of keys
   - Display product catalogs

5. **Exercise 5: Forms & Controlled Components**
   - Create controlled form inputs
   - Handle form submission
   - Validate user input
